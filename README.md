# KAST Tools 

🔗 **Live Demo**: https://kast-lite.vercel.app/

## ✨ Features

### 📊 Yield Optimizer
- **APY Comparison**: View demo yields from Gauntlet-powered vaults
- **Compound Interest Calculator**: Visualize growth over 1/5/10 years
- **Risk-Adjusted Returns**: Factor in risk coefficients for realistic projections
- **Rebalancing Simulator**: See how periodic rebalancing boosts long-term returns
- **Export Charts**: Download yield graphs as PNG

### 🌍 Payment Comparator
- **170+ Countries**: Search and filter supported destinations
- **Live Currency Conversion**: Real-time rates via exchangerate-api.com
- **Speed Comparison**: Solana (~5 sec) vs SWIFT (1-3 days)
- **Fee Calculator**: Compare $0.00025 (KAST) vs $25+ (traditional banks)
- **Savings Preview**: Estimate annual savings with KAST Pay
